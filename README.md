# VoidAI

VoidAI is a futuristic desktop AI assistant built with Electron, React, TypeScript, and Tailwind CSS.

## Development

Install dependencies:

```bash
npm install
```

Run development mode:

```bash
npm run dev
```

Build production files:

```bash
npm run dist
```

## Roadmap

- Phase 1: Desktop foundation
- Phase 2: Voice system
- Phase 3: Vision engine
- Phase 4: Memory
- Phase 5: Windows and browser assistance
